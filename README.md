# Lo que falta

**No se trata de contar. Se trata de lo que ya no está.**

Este proyecto propone una experiencia interactiva basada en la percepción y la ausencia. A partir de imágenes familiares intervenidas, el espectador es invitado a observar, contar y, finalmente, confrontar la desaparición como proceso. La obra busca desplazar la discusión del número hacia la experiencia: lo que ocurre cuando alguien deja de estar. No se trata de una cifra, sino de un vacío que persiste en quienes quedan.

## Tecnologías

* p5.js
* JavaScript

## Uso

1. Clonar o descargar el repositorio
2. Abrir el archivo `index.html` en un navegador
3. Interactuar tocando o haciendo clic en la pantalla para avanzar

Para una mejor experiencia, se recomienda visualizar en pantalla completa.
